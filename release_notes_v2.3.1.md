# GT23 Workflow v2.3.1 更新说明 (Release Notes)

---

## ⚡ 预览体验优化 (Preview Optimization)
- **即时反馈**：由于优化了底层的内存处理机制，现在调整边框参数（如边距、字体大小）时，预览图几乎没有延迟，能做到即点即看。
- **画质对齐**：修复了预览图和最终导出稿阴影不一致的问题，现在预览时看到的阴影效果就是最终成片的样式。

## 📂 批量处理改进 (Batch Processing)
- **有序命名**：在“马卡龙”和“彩虹”模式下，导出的文件名会自动加上 `001_`, `002_` 等序号前缀。如果您手动调整了样片的先后顺序，导出的文件也会按这个顺序排列，方便直接分享或排版。

## 📦 资源库更新 (Logo Library Update)
- **型号扩容**：同步更新了数十款相机 Logo（总计 200 余款），增加了对索尼 A7CII, A7R5, 徕卡 M11-P/D, M10-R 等机型的支持。
- **自动识别**：只要您的照片包含 EXIF 信息，现在程序能更精准地自动匹配这些最新热门机型的 Logo。
- **交互优化**：手动选择胶片时，在下拉框中选择或输入后按回车，预览图都会自动刷新。

## 📸 EXIF 元数据录入 (EXIF Metadata)
- **原图保留**：导出 JPEG 图片现在能完整保留原图的 EXIF 信息（如日期、地理位置等），不再是“白板”文件。
- **手动覆盖**：支持通过侧边栏手动改写品牌、型号、镜头、快门、光圈、ISO 等参数，并精准注入到图片底层。
- **全局应用**：新增“全局应用”开关。开启后，填写的参数会同步给当前批次内的所有图片，方便批量处理。

## 🛠️ 其他修复 (Bug Fixes)
- **稳定性修复**：修复了在 Windows 某些环境下程序可能报错导致无法运行的问题。
- **列表删除修复**：修复了在预览条中删除图片后，导出列表未同步更新的 Bug。
- **内存优化**：微调了批量导出时的处理逻辑，运行更稳。
