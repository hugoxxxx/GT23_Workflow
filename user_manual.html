GT23 胶片工作流

完整使用说明书 (v2.3.1)

欢迎使用 GT23 胶片工作流 (GT23 Film Workflow)。本工具专为胶片摄影师设计，旨在提供专业、优雅的影像后期呈现方案，包括全卷底片索引生成与高度自定义的影像边框处理。

1. 快速入门 (Quick Start)

目录与资源准备

程序首次运行会自动在根目录下创建以下文件夹：

  • photos_in/: 存放待处理的原始照片（支持 JPG/PNG）。
  • photos_out/: 处理后的成品照片将保存在此。
  • GT23_Assets/: 核心配置与资源目录，包含相机品牌图标、字体及配置文件。
提示： 首次运行建议点击 [帮助] -> [同步边框资源]。系统将从云端下载数以百计的相机 Logo、镜头配置文件及胶片预设。

2. 边框工具 (Border Tool)

此面板提供专业、精美的影像边框和 EXIF 数据标签。具备强大的自动识别与手动覆盖功能。

影像管理

  • 样片导航条：实时预览待处理图片，支持鼠标拖拽排序。
  • 工作模式：
    • 胶片项目: 侧重显示胶片品牌，自动开启颗粒质感。
    • 数码项目: 显示拍摄参数（ISO、焦距等），侧重器材展示。
    • 纯净模式: 仅保留边框线条，不添加任何文字。

高级设置

  • 边框主题: 提供 Default (浅色), Dark (深色), Macaron (马卡龙), Rainbow (彩虹) 等多种艺术主题。
  • 比例微调: 自由控制四周留白的厚度与字体比例。

EXIF 元数据录入

  • 原图保留: 导出时完整保留拍摄日期、GPS 等关键信息。
  • 全局应用: 一键同步手动输入的品牌、镜头信息至全批次。

3. 底片索引 (Contact Sheet)

专为“全卷底片”设计的排版工具，将多张样片自动排版成一张索引页。

  • 智能格式识别: 支持自动识别 135、120 (6x6, 6x7, 6x9) 等主流底片格式。
  • 乳剂号 (Emulsion No.): 允许为每卷胶片设置唯一的批次喷码。
  • 有序排列: 严格遵循文件名或拍摄时间的逻辑顺序。

4. 系统与资源

  • 双语切换: 全局支持中英文界面与输出文本。
  • 多源同步: 支持通过 Gitee 或 GitHub 更新相机资源库。
