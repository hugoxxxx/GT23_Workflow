# GT23 Film Workflow v2.3.0 发布说明

## 🎨 边框系统的细节进化 (Border System Refinements)

在 v2.3.0 版本中，我们对边框渲染进行了深度的稳定性加固与交互升级。这些变化将直接提升您在处理大批量照片时的效率与视觉一致性。

### 1. 彻底解决色彩跳变问题 (Enhanced Stability)
针对之前版本中“马卡龙”主题在预览或修改文字时颜色会随机变化的问题，我们实施了全新的底层逻辑。
- **色彩锁定**: 现在，每一张图片都会根据其物理位置和文件特征被赋予一个确定的色彩。无论您如何切换预览、修改 EXIF 参数，背景颜色都将保持稳定，不再“乱跳”。
- **重排同步**: 当您在缩略图栏通过拖拽调整照片顺序时，背景色会随位置实时更新，确保最终朋友圈“九宫格”排版色彩协调。

![Macaron Theme Preview](file:///d:/Projects/GT23_Workflow/previews/v2.3/macaron_preview.jpg)
*马卡龙主题：色彩与物理位置锁定，支持九宫格排版。*

### 2. 彩虹主题的连贯长卷 (Continuous Rainbow)
我们将“富士彩虹”重构并更名为更简洁的 **“彩虹” (Rainbow)**。
- **丝滑连拼**: 批量导出多张照片时，系统会自动计算并分配完整的光谱色彩。在将这些照片横向或纵向拼接时，您会看到一个丝滑、连贯的彩虹渐变长卷。
- **饱和度校准**: 优化了色彩渲染模型，解决了部分设备下色彩偏灰的问题，还原通透的胶片色彩感。

![Rainbow Theme Preview](file:///d:/Projects/GT23_Workflow/previews/v2.3/rainbow_preview.jpg)
*彩虹主题：多图连拼时的连续光谱效果。*

### 3. 单张图片的独立配置记忆 (Per-Image Persistence)
为了解决“牵一发而动全身”的痛点，我们实现了配置项的独立持久化。
- **独立开关**: 每一张照片的 EXIF 字段（制造、型号、镜头、快门、光圈、ISO）可见性现在是独立保存的。
- **互不干扰**: 您可以为某张图片特意隐藏镜头型号，而保留其他图片的完整信息。切换图片时，系统会自动恢复该图片对应的专属设置。

### 4. 极简主义与中文字符支持
- **纯净模式**: 新增“纯净模式”开关。勾选后，系统将全量滤除相机信息、参数文本与 Logo，仅保留边框底色。
- **中文排版**: 内置了中文字符自动检测逻辑。若您手动输入的字段包含中文（如“徕卡”、“哈苏”），渲染引擎将自动调用系统字库进行补齐，告别乱码。
- **即时预览**: 为所有参数输入框绑定了回车键，修改完成后“回车即看”，无需额外点击。
